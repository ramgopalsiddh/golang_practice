About Our Web App

Our web app leverages the Zomato Weather Union's API to provide users with accurate weather information. Utilizing latitude and longitude coordinates, but we retrieve data from table use city name & locality, the Weather Union's API to offer real-time weather updates for various locations.

To streamline the process, we've established a localities database table. This table stores essential information including city name, locality name, latitude, longitude, and device type. The localities table serves as a mapping tool, allowing users to easily access weather data by specifying the city and locality name.
